# 株式会社ケーイーシー 公式サイト

三菱系FA機器・工作機械の電気工事を専門とする株式会社ケーイーシーの企業公式サイトです。

## 概要

このサイトは、愛知県稲沢市に本社を置く株式会社ケーイーシー（法人番号: 8180001085783）の公式ウェブサイトです。工場のFA機器や工作機械の電気工事に強みを持つ企業として、サービス内容、施工事例、会社概要などを掲載しています。

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **ビルド出力**: Static Export

## プロジェクト構造

```
KEC-Official-Site/
├── app/                      # Next.js App Router
│   ├── page.tsx             # トップページ
│   ├── about/               # 会社概要ページ
│   ├── services/            # サービス・強みページ
│   ├── case-studies/        # 施工事例ページ
│   ├── contact/             # お問い合わせページ
│   ├── layout.tsx           # ルートレイアウト
│   └── globals.css          # グローバルスタイル
├── components/              # 共通コンポーネント
│   ├── Header.tsx           # ヘッダー
│   └── Footer.tsx           # フッター
├── public/                  # 静的ファイル
├── package.json            # 依存関係
├── tsconfig.json           # TypeScript設定
├── tailwind.config.ts      # Tailwind CSS設定
└── next.config.ts          # Next.js設定
```

## セットアップ

### 前提条件

- Node.js 18.17以降
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

開発サーバーが起動したら、ブラウザで http://localhost:3000 にアクセスしてください。

### ビルド

```bash
# 本番用ビルド（静的サイト生成）
npm run build

# ビルドされたファイルは out/ ディレクトリに出力されます
```

## Coming Soon 項目について

サイト内の「Coming soon」バッジが表示されている箇所は、クライアント様との商談後に実際の情報を追加する予定の部分です。

### 主なComing Soon項目

1. **企業情報**
   - 設立年月日
   - 資本金
   - 代表者名
   - 従業員数
   - 電話番号
   - メールアドレス
   - FAX番号
   - 営業時間

2. **コンテンツ**
   - 企業理念・ビジョン
   - 代表メッセージ
   - 会社沿革
   - 保有資格・認証
   - 技術力の詳細
   - 対応エリア
   - 対応時間
   - 価格情報

3. **施工事例**
   - 実際のプロジェクト写真
   - 施工内容の詳細
   - お客様の声
   - 成果データ

4. **機能**
   - お問い合わせフォーム送信機能

## プレースホルダーの実装方針

「Coming soon」バッジとプレースホルダーセクションには以下のスタイルクラスを使用:

```css
.coming-soon-badge {
  @apply inline-block bg-yellow-100 text-yellow-800 text-xs font-semibold px-2 py-1 rounded;
}

.placeholder-section {
  @apply border-2 border-dashed border-yellow-400 bg-yellow-50 p-6 rounded-lg;
}
```

これにより、クライアント様がどこに情報を追加する必要があるか一目で分かるようになっています。
